# Robovian

**Robovian** is an AI and Robotics learning app designed for students, beginners, and future robotics engineers.  
It helps users learn robotics, explore projects, understand components, and get instant guidance from an in-app AI assistant called **RoboGenie**.

---

## Features

### Main App Features
- Modern robotics-focused home screen
- AI and Robotics learning support
- Beginner to advanced project suggestions
- Daily challenges and learning tasks
- Progress tracking
- Category-based learning sections
- Premium futuristic UI

### RoboGenie AI Assistant
**RoboGenie** is the core AI assistant of Robovian.

It helps users with:
- AI and Robotics learning
- Python, Arduino, C/C++ guidance
- Project suggestions
- Code help
- Circuit and wiring help
- Components selection
- Humanoid robotics basics
- Roadmap generation
- Quiz and test preparation
- Daily challenge support

## Purpose of Robovian

The goal of Robovian is to make **AI and Robotics learning simple, smart, and practical**.  
Instead of only showing theory, the app guides users through:
- concepts
- projects
- coding
- hardware understanding
- AI-based robotics thinking
